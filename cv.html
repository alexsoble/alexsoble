Student Insights for Somerville

Started in 2015, now we're here.

Open source technology to support educators in Somerville, MA.

Used in three elementary schools and counting.

Started as a Code for America fellowship project, now a mix of contract work and incredible open source contributions from Code for Boston members.

Rails back, React front.

Links data from Somerville's Student Information System with external assessment database plus data entry.

UI built for educators, with educators.

Take a look at the README for lots more information.
